# Auralis — Music Player

A premium, feature-rich web-based music player built with HTML, CSS, and JavaScript.

![Preview](preview.png)

## Features

- **Playback Controls** — Play, Pause, Next, Previous with keyboard shortcuts
- **Progress Bar** — Click or drag to seek through tracks
- **Volume Control** — Slider with mute/unmute states
- **Playlist** — Click any song to play, active track highlighting
- **Shuffle & Repeat** — 3-mode repeat (off / one / all)
- **Autoplay** — Automatically advances to the next track
- **Visualizer** — Animated frequency bars synced to playback
- **Toast Notifications** — Slide-up alert on track change
- **Keyboard Shortcuts**
  | Key | Action |
  |---|---|
  | Space | Play / Pause |
  | ← / → | Previous / Next |
  | ↑ / ↓ | Volume + / - |
  | S | Toggle Shuffle |
  | R | Cycle Repeat mode |

## Tech Stack

- HTML5 `<audio>` API
- CSS3 — Glassmorphism, keyframe animations, gradients, responsive design
- Vanilla JavaScript — No frameworks or libraries

No build tools or server required.
